# Steemit Upvote Bot


## Installation

    git clone https://mrMuffins@bitbucket.org/mrMuffins/steem-bot-upvote.git
    
    cd steem-bot-upvote

    sudo ./install.sh

    
This is where you will set the account, posting key, and users that you wish to upvote
    
    nano steemvars.py

    
    ./run.sh
